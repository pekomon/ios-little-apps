# TicTacToe

`TicTacToe` is a small SwiftUI game project.

## What the app includes

- 3x3 game board
- Turn-based X/O gameplay
- Win detection
- Winning-cell highlighting
- Restart action
- Basic animations and dark mode support

## Learning focus

- SwiftUI state handling
- MVVM-style project structure
- Observable objects and published state
- Simple game logic
- Unit tests and UI tests

Finished small learning project.
