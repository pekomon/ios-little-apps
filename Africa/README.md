# Africa

`Africa` is a SwiftUI learning project focused on browsing content about African animals.

## What the app includes

- Animal list and grid views
- Detail views for animals
- Video list
- Map/location view
- Gallery tab
- Custom SwiftUI views and modifiers

## Learning focus

- `NavigationView`
- `TabView`
- `List`, `ScrollView`, and `LazyVGrid`
- Loading bundled JSON data
- Reusable SwiftUI components
- Basic haptics and simple animations

## Status

Finished learning project.
