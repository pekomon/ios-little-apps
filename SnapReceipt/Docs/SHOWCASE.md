# SnapReceipt Showcase Assets

This folder holds the current SnapReceipt presentation assets used for the project README and repository showcase.

## Screenshots

All screenshots were captured from the deterministic demo-launch states on iPhone 17 simulator builds.

- `Screenshots/capture.png`: import state with OCR summary and readiness panels
- `Screenshots/review.png`: editable review flow before save
- `Screenshots/receipts.png`: saved receipts list with local-history framing
- `Screenshots/settings.png`: local-first settings and storage details

## App Icon Sources

- `icon-standard.svg`

- `icon-tinted.svg`

## Re-capturing The Set

Use the `SNAPRECEIPT_DEMO_SCENARIO` launch environment to open stable app states:

- `capture`
- `review`
- `receipts`
- `settings`

This keeps the README and repository showcase screenshots visually consistent across future refreshes.
